Mini Pokédex hecha con SwiftUI (MVVM).
API: PokeAPI
- Lista: https://pokeapi.co/api/v2/pokemon?limit=50
- Detalle: https://pokeapi.co/api/v2/pokemon/{name}

Funcionalidad:
- Lista de los primeros 50 Pokémon
- Pantalla de detalle con sprite, tipos, altura y peso
- Loading, pull-to-refresh y manejo de errores
